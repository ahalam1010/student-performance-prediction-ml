# Student Performance Prediction Using Machine Learning

## Overview
Predict student dropout risk using supervised machine learning models to support early academic intervention.

## Dataset
- 649 student records
- 33 features
- Preprocessing and feature engineering applied

## Methods
- PCA retaining 95% variance
- Class imbalance handled using SMOTE
- Models: Random Forest, Neural Network

## Results
- Accuracy: ~85%
- Dropout recall improved to 50%

## Tools
Python, scikit-learn, PCA, SMOTE
